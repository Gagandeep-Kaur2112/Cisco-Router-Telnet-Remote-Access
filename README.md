# Cisco Router Telnet Remote Access

## Overview
This project demonstrates remote management of a Cisco router using Telnet in Cisco Packet Tracer. The objective was to configure secure remote login access and test connectivity from a remote device.

## Technologies Used
- Cisco Packet Tracer
- Telnet
- Cisco IOS CLI

## Configuration Highlights
- VTY line configuration
- Password authentication
- Remote login testing via Telnet
- Basic IP connectivity setup

## Skills Demonstrated
- Router configuration
- Remote administration
- Command-line interface usage
- Network troubleshooting
